GUGU (Windows)
==============
화면에 비둘기가 마우스 커서를 따라다니며 쪼아대는 데스크탑 오버레이.


요구 사항
---------
* Windows 10 / 11 (64-bit)


설치
----
1. GUGU-1.0-Windows.zip 압축 풀기 → GUGU 폴더가 생깁니다.
2. GUGU 폴더는 통째로 한 곳에 두세요 (예: 바탕화면, 문서 폴더).
   안에 든 GUGU.exe 만 따로 꺼내면 안 됩니다.
3. GUGU.exe 더블클릭으로 실행.

   ※ "Windows의 PC 보호" (SmartScreen) 경고가 뜨면:
      - "추가 정보" → "실행" 클릭
      - 서명되지 않은 앱이라 뜨는 안내일 뿐, 한 번만 허용하면 됩니다.


사용법
------
* 실행하면 작업 표시줄 우측 트레이에 비둘기 아이콘이 뜹니다.
* 마우스 커서 옆에서 비둘기가 따라다니며, 멈추면 쪼아댑니다.
* 다른 앱을 클릭해도 비둘기는 계속 화면에 떠 있습니다.


종료 방법
---------
1. Ctrl+Alt+Q  — 글로벌 단축키, 어디서나 동작
2. 트레이 비둘기 아이콘 우클릭 → 종료
3. 작업 관리자에서 "GUGU" 프로세스 종료


설정 (트레이 메뉴)
------------------
* 클릭 통과 (Click-through): 기본 ON
  - ON  : 비둘기 위로 마우스 클릭이 통과 → 작업 방해 없음
  - OFF : 비둘기를 클릭하면 푸드덕 날아오름


바로가기 만들기
---------------
1. GUGU.exe 우클릭 → "바로 가기 만들기"
2. 만들어진 바로 가기를 바탕화면이나 시작 메뉴로 이동.

시작 시 자동 실행 (선택)
------------------------
1. Win+R → "shell:startup" 입력 → Enter
2. 열린 폴더에 GUGU.exe 바로가기를 복사해 넣으면 부팅 시 자동 실행.
